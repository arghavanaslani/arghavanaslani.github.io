## Summary

This paper challenges the classical attractor model of working memory, which assumes that information is held in memory via sustained elevated firing rates in prefrontal neurons. Lundqvist et al. instead show — using recordings in macaque PFC — that working memory is associated with *bursts* of gamma oscillations (40–100 Hz) and *decreases* in beta oscillations (10–30 Hz), without requiring persistent spiking activity.

The key idea is that memory items can be stored in short-term synaptic weights (activity-silent storage), with gamma bursts serving to "read out" or refresh information when needed. This is more energetically efficient and consistent with observed neural dynamics.

## Questions

### Why would gamma bursts be preferable to sustained firing for memory maintenance?

Sustained firing is metabolically expensive and prone to noise accumulation over time. Gamma bursts allow for periodic refreshing of information held in synaptic weights, reducing the energy cost while maintaining robustness.

### What is the relationship between beta suppression and gamma increases?

Beta suppression appears to gate the network into a state where gamma activity can emerge — when beta is high, the network is in a "default" or inhibited state; suppressing beta releases this inhibition and allows gamma bursts to occur. Still reading into the exact mechanism.
